Webscraping - MENA Newsletter




This is a complete data engineering process. The idea is to scrape the latest news title from different websites including BBC, Wall Street Journal, Al Jazeera, Reuters, Financial Times, Bloomberg, CNN, Fortune and The Economist. The raw data then goes through a few pipelines to be cleansed and stored in a database. Due to unique value constrain of the database, only latest news would be embed into HTML email. Eventually HTML emails are automatically sent out everyday as newsletter.

Click here to view the first version of this project. The biggest difference between two versions is the process order. For the first version, the order is scraping the first website, ETL, inserting into database, getting output and repeating the process. For the second version, the order is scraping all the websites, concatenating them, ETL, inserting into database and getting output. The advantage of the latter version is that it can be integrated with Graph Theory - Text Mining to remove similar content and extract the key information. In addition, when encountering network issues in the middle of scraping, we simply reboot the script without worrying over inserted data of the previous websites. Other minor improvements include adding preview image to HTML email and using dataframe output rather than list output. As for optimized Naive Bayes, both versions of web scraping can be easily integrated into NLP framework.
